shouldComponentUpdate 方法和 React.memo 方法什么时候用，区别是什么？

shouldComponentUpdate 一般用于比较 state 或 prop 有没有变化，有变化就 update，没变化就不 update，但是！！，这个功能函数组件已经替我们实现了！！（使用 hook 的好处之一？）
